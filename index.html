Что такое HTML

HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный язык разметки документов во Всемирной паутине. Большинство веб-страниц имеют содержание разметки на языке HTML (или XHTML). Язык HTML интерпретируется браузерами; полученный в результате интерпретации форматированный текст отображается на экране монитора компьютера или мобильного устройства.

Теги для оформления текста

Strong - выделяет текст жирным начертанием. Помимо визуально эффекта, имеет смысловое (семантичское) значение. Текст выделенный таким тегом важен по смыслу, при чтении на нем стоит сделать акцент, подчеркнув его значимость.

Bold - выделяет текст жирным начертанием. Имеет только визуальную нагрузку.

Em - Emotional. Выделяет текст курсивом. Имеет смысловое значение.

I - Italic. Выделяет текст курсивом. Имеет только визуальную нагрузку.

code - используется для обозначения кода в тексте. Например: let x = 1;
